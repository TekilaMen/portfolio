Ecolotri

Html / Php / Css / SQL

Mise en situation

Ecolotri est un projet mené en classe et ensuite finalisé chez moi, voici le contexte :
La société Ecolotri a pour activité de traiter et valoriser les déchets en provenance des déchèteries de la région. Chaque déchetterie est gérée par un syndicat qui est un regroupement de communes. Le principe est simple, lorsqu'un camion se présente à l'accueil de l'entreprise, il doit préciser le syndicat pour lequel il travaille, fournir son immatriculation et le type de déchet transporté (verre, métal, carton, bois, végétaux, ménager…). Il est pesé avant d'aller décharger sa marchandise. En repartant il est à nouveau pesé, ce qui permettra par différence de connaître le poids de déchet déposé (l'unité utilisée est toujours la tonne).
Vous trouverez ci-dessous le MCD (Modèle conceptuel de données) :

Cahier des charges

Le cahier des charges nous demande de développer une application web simple permettant :
- De saisir une nouvelle pesée,
- D'afficher la liste des pesées par date heure décroissante, et sur un clic d'une pesée d'afficher le détail,
- D'afficher diverses statistiques.

En respectant certaines attentes qui sont composées de :
- Un script SQL qui permet de créer la base de données et d'insérer les types de déchet et les syndicats après avoir mené une réflexion sur les types de données,
- Une interface simple et efficace,
- Une application fonctionnelle en 3 séances.

Avis & Finalité

Ce projet m'a permis de m'initier sérieusement au PHP, en effet étant l'un de nos premiers projets cela nous a donc fourni les fondements de ce langage tel que l'utilisation de la méthode POST et GET. Cependant, malgré un groupe composé de trois personnes, nous n'avons pas pu finaliser ce projet en classe dans les temps. J'ai donc pris sur mon temps libre pour le finaliser avec l'aide d'un de mes camarades.
